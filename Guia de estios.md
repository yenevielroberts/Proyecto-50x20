# Guia de estilos

## Colores

### Colores principales:

- Azul suave (#4A90E2) → Confianza + calma
- Verde esperanza (#4CAF50) → Bienestar + apoyo
- Turquesa claro (#5AD4CE) → Juventud + frescura

### Colores Secundarios:

- Amarillo pastel (#F7E48B) → Amabilidad + calidez
- Naranja suave (#F79C42) → Llamadas a la acción
- Gris muy claro (#F5F6FA) → Fondo suave
- Gris oscuro (#263238) → Texto principal (mejor que negro puro)

Titulos: #4A90E2
